# Hệ thống quản lý và phân tích dữ liệu tội phạm TP. Hồ Chí Minh

Ứng dụng console bằng C# để quản lý và phân tích dữ liệu vụ án tại Thành phố Hồ Chí Minh.

## Tính năng
- Thêm, xem, tìm kiếm và xóa vụ án.
- Lưu dữ liệu thành file JSON.
- Phân tích số vụ theo quận/huyện, loại tội phạm và tháng.

## Cách chạy
1. Mở terminal tới thư mục dự án.
2. Chạy `dotnet run`.

Dữ liệu sẽ lưu trong file `crime_data.json`.
